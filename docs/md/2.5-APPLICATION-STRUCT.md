目录结构
=============================

```php
data-dictionary             根目录
|----docs                   文档目录            
    |----md                 开发者文档   
|----lib                    第三方源码存放目录(企业微信官方加密解密)
|----src                    源码存放目录
    |----Contacts           通讯录操作(联系人、部门api等)
    |----Events             应用事件
    |----Exceptions         异常类
    |----Helpers            常用助手函数
    |----Login              登录(单点登录和oauth2登录)
    |----Suites             企业微信服务商 api
|----tests                  单元测试存放目录
|----vendor                 composer安装依赖包目录

```
